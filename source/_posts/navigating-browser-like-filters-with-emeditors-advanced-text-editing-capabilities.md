**Find** toolbar -> “Browse Find (or Replace) Expressions” button,  
 Select a regular expression or escape sequence from the list, it will insert the selected item into the text box.  
 \=> OK  
**Filter** toolbar -> “Browse Filter Expressions” button,  
 Select a regular expression or escape sequence from the list, it will **Not** insert the selected item into the text box.  
 \=> The behavior is different than Find toolbar, Find and Replace dialog box.  
After checked the older versions(v14.x), “Browse Filter Expressions” button will insert the selected item.
